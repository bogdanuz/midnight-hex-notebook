# Changelog

## [1.0.0] — 2026-03-02

- Первый релиз нового репозитория `midnight-hex-notebook`.
- Локальный генератор UDP CPS-пакетов (`GenSpecialJunkPacket.py`) и модули в `src/` для AWG 1.5/2.0.
- Статический сайт в `web/` для выдачи I1–I5 в формате `<b 0xHEX>`, совместимый с форматом CPS.
- Автотесты (pytest) для крипто, генерации пакетов, конфига и структуры web-страницы.
- Базовый Makefile, pyproject, .gitignore и пример `.env.example` для локального использования.
