# Midnight Hex Notebook

> Локальный генератор и веб-раздача значений I1–I5 в формате hex для подстановки в клиенты с поддержкой CPS.

## Что это

- **Сайт** (`web/`): раздаёт готовые наборы I1–I5 из JSON. Открываешь `web/index.html` или деплоишь папку `web/` на статический хостинг.
- **Скрипт** (корень): генерирует UDP-пакеты, похожие на SIP/TLS/HTTP, проверяет доставку, сохраняет результат в JSON. Запуск локальный; на хостинге Python не выполняется.
- **Модули** (`src/`): валидация конфига, генерация пакетов и паддинга. Используются в тестах.

Данные подходят для клиентов, где в настройках протокола есть поля I1–I5 (формат CPS). Остальные параметры (H1–H4, S1–S4, Jc, Jmin/Jmax) задаются в самом клиенте.

## Структура

```
├── web/                 # Сайт: index.html, I-list1.json
├── docs/                # Локальная инструкция для агента (AGENT_REFRESH_DATA.md)
├── audit/               # Эталонная спецификация AWG (AWG-SPEC-REFERENCE.md)
├── src/                 # Python: протокол, пакеты, крипто, конфиг
├── tests/               # pytest
├── .github/workflows/   # CI
├── GenSpecialJunkPacket.py
├── pyproject.toml
├── Makefile
└── README.md
```

## Возможности

- Выдача I1–I5 через веб (папка `web/`), копирование в буфер.
- Генератор: SIP REGISTER, TLS Client/Server Hello, HTTP over TLS; проверка доставки, сохранение в JSON.
- Модули в `src/`: конфиг, валидация, junk-пакеты.

## Требования

- Сайт: любой современный браузер.
- Генератор: **Python 3.8+** (stdlib).
- Тесты: **Python 3.11+**, `pip install -e ".[dev]"`.

## Установка

```bash
git clone https://github.com/ТВОЙ_НИК/midnight-hex-notebook.git
cd midnight-hex-notebook
make install
```

## Запуск

- **Сайт:** открыть `web/index.html` или залить `web/` на хостинг (GitHub Pages: источник — папка `web/`).
- **Генератор:** `make run` или `python GenSpecialJunkPacket.py`. Результат — `packets.json` в корне; для сайта скопировать в `web/I-list1.json`.
- **Тесты:** `make test`.

## Конфигурация

Скопировать `.env.example` в `.env` при необходимости (хост/порт для UDP-теста генератора).

## Совместимость с клиентами

Значения I1–I5 в формате `<b 0xHEX>` совместимы с протоколами 1.5 и 2.0. В приложении подставь их в поля «Special junk packet» (I1–I5) по инструкции в модальном окне на сайте.
